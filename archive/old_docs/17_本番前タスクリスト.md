## 🆘 トラブル時の対応

### VPN がつながらない

1. プロトコル種別を確認（L2TP/IPsec? PPTP?）
2. PSK 入力ミスを確認
3. ID/Pass 入力ミスを確認
4. 別 ID（booth12 等）で試す
5. 運営に確認

### SSH がつながらない

1. VPN 状態確認
2. IP 間違いないか（10.1.1.1 / 10.1.1.2）
3. `ping` で疎通確認
4. ファイアウォール（自分の PC 側）確認

### PC が固まる / 落ちる

1. 焦らない
2. チームの別メンバーに状況共有
3. 再起動
4. それでもダメならスマホでテザリング → 別 PC（あれば）

### 自分が冷静になれない

1. **席を立って深呼吸 3 回**
2. 水を飲む
3. チームに「ちょっとリセットさせて」と声をかける
4. **判断を先延ばし** にする（迷ったら何もしない）
